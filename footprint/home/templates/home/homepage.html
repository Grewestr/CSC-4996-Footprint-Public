Quickly Locate Missing People Using Computer Vision

Footprint is a system designed to rapidly locate missing individuals by analyzing live video feeds. Utilizing existing technology, it recognizes people based on clothing and appearance without the need for specialized equipment.

Problem & Solutions

Identifying people based on clothing and appearance poses challenges due to frequent outfit changes and environmental factors like poor lighting or weather conditions. Footprint addresses these issues by combining features such as color and texture with time and location data.

Why Choose Footprint?

  • Fast and Efficient
  • Cost-Effective
  • Advanced Technology

Target Audience

Ideal for law enforcement agencies, search and rescue teams, and public safety organizations.

Technologies We Use

  • YOLOv5 for object detection
  • PyTorch for model training
  • OpenCV for image processing
  • Django for backend development
  • Python for functions
  • Firebase for storing attributes and managing queries
  • GitHub for version control
